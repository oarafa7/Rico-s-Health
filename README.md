# RICO - Stack Tracker

Track peptides, hormones, supplements, and nutrition with reconstitution math, vial tracking, and macro counting.

## Deploy (free, 15 minutes)

1. Create a GitHub repo and push this folder
2. Go to vercel.com, sign in with GitHub
3. Import the repo, click Deploy
4. Open the URL on your phone in Safari → Share → Add to Home Screen

## Local dev

```
npm install
npm run dev
```

## Data & Privacy

All data stored locally on device. No server, no tracking, no analytics.
